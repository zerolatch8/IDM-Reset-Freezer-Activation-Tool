# IDM Tool - Unified Configuration Manager v1.0.0

**Developed by ZeroLatch | Credits: ZIEDEV**

A professional, modular Windows desktop application to configure, preserve, and manage Internet Download Manager (IDM) with ease.

---

## 🚀 Features

- **Permanent Configuration** - Registers IDM with a permanent registration token.
- **Preserve Evaluation** - Stops the 30-day trial timer permanently without requiring a Token.
- **Reset IDM** - Completely clears all IDM registration data and trial history.
- **Quick configure** - Instantly applies a random Token directly to the registry.
- **Status Monitoring** - Real-time check of your IDM registration state.
- **Automatic Cleanup** - Cleans existing host file blocks before operation to ensure a fresh start.

---

## 📦 Installation & Usage

1. **Download** the latest release from the [Releases](https://github.com/zerolatch8/IDM-Reset-Freezer-Activation-Tool/releases/latest) page.
2. **Extract** the ZIP file to a folder of your choice.
3. **Run** `IDM_Tool.exe` as **Administrator**.
4. Use the intuitive GUI to select your desired operation.

> [!IMPORTANT]
> Ensure IDM is closed (including the system tray icon) before performing any Configuration or reset operations.

### 🛡️ Windows SmartScreen Warning
Because this tool is relatively new and unsigned, Windows may display a **"Windows protected your PC"** blue warning screen when you first open it. 
To bypass this and run the tool safely:
1. Click on **More info** (the underlined text under the main warning).
2. Click the **Run anyway** button that appears at the bottom right.
